# Java-Web-Application-Development
## Technique used : 
+ MVC , MVC2 Pattern
+ Servlet
+ JDBC
+ Dispatcher
+ Dynamic DB Connection
+ Error Handling in Servlets
+ Sessions and Listeners
+ Expression Language
+ JSTL (JSP Tag Libraries )
+ Filter
## Necessary 
+ sqljdbc4.jar
+ Netbeans 8
+ ApacheTomcat 8.5.29
+ JSTL 1.2.1
+ SQL 2019
